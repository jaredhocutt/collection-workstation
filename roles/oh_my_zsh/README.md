Oh My Zsh
=========

Installs oh-my-zsh and sets the user's default shell to zsh.

Requirements
------------

- python >= 2.6
- python-dnf

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

```yaml
- hosts: servers
  roles:
    - role: oh_my_zsh
```
